# BiteBalance Landing Page

A clean, responsive single-page landing site for BiteBalance, a modern diet and nutrition tracking app.

## Overview
This project is a static promotional webpage designed to convert visitors into app users. It uses a semantic structure, a custom earthy color palette, and a modern side-by-side feature card layout.

## Features
* Semantic HTML: Accessible structure using semantic tags (header, nav, section, footer).
* Responsive Design: Custom CSS Flexbox and Grid layouts that adapt to desktop, tablet, and mobile screens.

## Technologies Used
* HTML
* CSS

Note: I used AI to assist with the ideation and scaffolding of this project. The AI suggested the app name ("BiteBalance"), the hero copy, and the color palette. I wrote and structured the HTML/CSS and implemented the layout logic.

## Setup
1. Clone this repository.
2. Open index.html in any modern web browser. No local server or build tools are required.
